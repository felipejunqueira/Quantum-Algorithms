# Quantum Algorithms

Some simple codes written in Python, using the libs from Qiskit and friends.

## Implemented Algorithms and Concepts
* **EstadoBell:** Complete implementation of fundamental entanglement, including the 4 Bell States, GHZ State, and CHSH Inequality test.
* **Grover:** Implementation of Grover's unstructured search algorithm with custom Oracles and Diffusers.
* **QFT:** Quantum Fourier Transform, demonstrating phase encoding and geometric bit-reversal correction (SWAP).
* **QPE:** Quantum Phase Estimation, demonstrating phase kickback and eigenvalue extraction using $QFT^\dagger$.
